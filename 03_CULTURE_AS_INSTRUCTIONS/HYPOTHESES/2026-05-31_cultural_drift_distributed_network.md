## One-Line Claim

Cultural drift is the movement of instruction-variant distributions inside a
distributed network of interpreters, artifacts, institutions, and interpreted
objects.

## Why This Exists

The current Culture canon already says:

```text
culture = externalized network of instructions
```

Existing CID and operator-dynamics files add decay, persistence, spread, and
mutation.

This hypothesis adds the missing macro-dynamics layer:

```text
how cultural variants move, attract, diverge, stabilize, and become artifacts
or institutions across domains
```

## Core Model

Culture is not a sequence of pure ideas.

It is a distributed network where every idea exists as a cloud of variants:

```text
IdeaCloud = variants + weights + carriers + contexts + artifacts + competing
interpretations
```

Cultural drift is the change in that cloud over time.

The drift may be random, but it is not only random. It is biased by:

- carrier strength;
- problem-fit;
- emotional weight;
- artifact durability;
- copying cost;
- institutional support;
- resource constraints;
- group identity needs;
- compatibility with existing instruction complexes;
- collisions with rival idea clouds.

## Minimal Object Model

### Interpreters / carriers

People and groups mutate, reject, combine, simplify, and transmit idea variants.

### Artifacts

Artifacts stabilize variants outside individual memory.

They include tools, images, buildings, rituals, texts, songs, clothing forms,
laws, interfaces, institutions, and other external carriers.

### Objects of interpretation

Culture also forms around what is interpreted:

- nature;
- bodies;
- other groups;
- social events;
- technical constraints;
- death, illness, weather, scarcity, danger, and anomaly.

The same object can generate different idea clouds in different networks.

## Dynamic Operators

The useful operators are not isolated metaphysical forces. They are interaction
modes inside the network.

| Operator | Network Reading |
|---|---|
| Variation | Every cultural idea exists as competing local variants. |
| Negation | A variant defines itself against a prior attractor or rival cloud. |
| Attraction | Nearby variants increase each other's activation or copying probability. |
| Synthesis | Variants combine into a new stable instruction complex. |
| Artifact fixation | A variant becomes easier to copy because it is stored in an external carrier. |
| Institutional fixation | A variant is backed by roles, sanctions, repetition, and training. |
| Drift | The distribution changes through mutation, copying, forgetting, and selective retention. |
| Decay | A variant loses weight when reproduction and carrier support weaken. |

## Cohesion Artifact Claim

Art, religion, law, ritual, and identity symbols can function as cohesion
artifacts when they reduce group fragmentation or increase coordination.

This is a functional claim, not a reduction:

```text
cohesion function != full explanation of art, religion, or law
```

The hypothesis should remain careful around religious, artistic, and historical
material.

## Sedentism Simulation Lead

The source conversation sketches a conceptual simulation:

```text
people + groups + resources + artifacts + storage + cooperation + cohesion
instructions -> nomadic / settled stability outcomes
```

This is useful as a model-design lead, but not as archaeology evidence.

Possible variables:

- resource density;
- resource renewal rate;
- storage capacity;
- group size;
- copying fidelity;
- artifact durability;
- split probability;
- cooperation/egoism balance;
- cohesion artifact strength;
- conflict or competition pressure.

## Predictions

1. Cultural domains with different media should show analogous cycles of
   variation, negation, synthesis, stabilization, and decay.
2. Artifacts with high problem-fit and low copying cost should stabilize
   variants more strongly.
3. Institutions should increase persistence but may reduce variant diversity.
4. High-spread variants should not automatically have high persistence.
5. Cultural jumps should often look like rising variant diversity followed by
   stabilization around attractors.
6. Group-scale symbolic systems should correlate with larger and more stable
   coordination networks only when supported by resources, repetition, and
   carrier infrastructure.

## Failure Modes

Demote or split the hypothesis if:

- it explains every cultural change after the fact and predicts nothing;
- it duplicates CID or MEAT without adding a distinct network-drift layer;
- historical cases are better explained by domain-specific constraints alone;
- the artifact/cohesion layer turns into reductionism;
- simulation language replaces source-backed anthropology;
- PNA residue is smuggled in as neuroscience evidence.

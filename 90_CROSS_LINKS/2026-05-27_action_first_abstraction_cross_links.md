# Action-first abstraction cross-links

## Status

Status: `candidate_support`.

This cross-link note supports the R Theory candidate patch:

- `01_R_THEORY/PATCHES/2026-05-27_action_first_abstraction_candidate_patch.md`

It does not update canon.

## Core bridge

Candidate formula:

```text
operation -> stabilized operation -> portable action category -> recombinable action grammar -> external trace -> symbolic carrier
```

This formula links R Theory to adjacent modules by placing a pre-symbolic action abstraction layer before symbolic carrier formation.

## Link 1: R Theory

R Theory owns the core distinction:

```text
artifact as object/form != artifact as external trace of stabilized operation
```

R Theory use:

- R1: repeated action;
- R1+/R2-: proto-operation;
- R2: portable action category;
- R2+: recombinable action grammar;
- R3: not established by this chain unless explicit comparison, causal rule correction, cross-domain transfer, or reflective error analysis appears.

Boundary:

This link does not promote Oldowan, Acheulean, ochre, or burial into canon-level R assignments.

## Link 2: Culture as Externalized Instructions

Culture as Externalized Instructions can use this patch to clarify how action routines become transmissible instructions.

Bridge:

```text
action grammar -> teachable operation -> copied instruction -> cultural tradition
```

Possible use:

- Oldowan / Acheulean are not merely objects copied by form;
- they may be copied operational sequences;
- an artifact tradition can preserve an instruction even when the learner never sees the original maker's mental model.

Boundary:

This does not prove that early tool traditions were symbolic culture in the strong sense. It only supports an intermediate layer where repeated operations can become transmissible action-instructions.

## Link 3: PNA / consciousness and action representation

PNA / action representation can use this patch as a lower-level bridge between neural action coding and later reflective cognition.

Bridge:

```text
neural action unit -> stabilized operation -> external trace
```

The PMv/action-symbol evidence suggests that an action can become invariant, categorical, and recombinable at the planning level before language-like symbolic systems.

Boundary:

This does not prove a full consciousness model, a PNA mechanism, or R3 self-reference. It only gives a candidate interface between neural action representation and operational abstraction.

## Link 4: Archaeology / artifact interpretation

The archaeology block gains a methodological rule:

```text
do not read artifacts only as objects; read them as traces of operations
```

Layered reading:

| Layer | Question |
|---|---|
| Morphology | What object/form is present? |
| Operation | What action sequence produced it? |
| Category | Is the operation portable across materials/context? |
| Grammar | Are operations recombined into ordered sequences? |
| Transmission | Is the sequence copied/taught/stabilized socially? |
| Significance | Does the trace/practice retain meaning beyond immediate function? |

Boundary:

Artifact form alone is insufficient. A strong claim needs evidence about operational sequence, repetition, context, and transmission.

## Link 5: Symbol-to-concept path

The existing R Theory v3.2 chain is:

```text
function -> significance -> symbol -> transferable distinction -> concept
```

Action-first abstraction adds a lower layer before that chain:

```text
operation -> action-symbol/action-grammar -> external trace -> function/significance -> symbol -> transferable distinction -> concept
```

Boundary:

The action layer does not automatically create a symbol or concept. It creates a structured trace that can later become a symbolic carrier if significance stabilizes.

## Misuse risks

1. Treating action-symbol as full symbol.
2. Treating action grammar as R3.
3. Treating artifact morphology as direct mental evidence.
4. Treating archaeology sequence as a universal ladder.
5. Collapsing operation, instruction, symbol, and concept into one step.

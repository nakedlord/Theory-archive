## What This Module Owns

This module owns the hypothesis that Nonviolent Communication can be interpreted as a practical continuation of a Kant-like discipline of judgment: separating what belongs to the object/event from what belongs to the subject's relation, affect, need, interpretation, and request.

Core operator:

```text
same grammatical predication shape
+ mixed judgment types
+ subjective relation projected as object property
-> conflict escalation / violent communication

predicate hygiene
+ FACT / INTERPRETATION / FEELING / NEED / REQUEST separation
-> nonviolent communication
```

## What It Does Not Own

- It does not claim that Kant invented NVC.
- It does not reduce Kant's critical philosophy to communication technique.
- It does not claim that all analytic or linguistic philosophy derives from Kant in this specific sense.
- It does not treat every negative judgment as violent communication.
- It does not deny that objects and people can have real properties or stable behavioral patterns.
- It does not canonize an interpretation of Kant without textual verification.

## Current Status

The module is a durable draft section created from a user-led discussion while reading Kant on pleasure, agreeable/beautiful distinctions, and the way language hides different operations under one predicate form.

The idea is worth preserving because it connects three existing lines of work:

1. Kant's hidden language hygiene: grammar can mask distinct judgment operations.
2. SynTax level discipline: FACT / INTERPRETATION / HYPOTHESIS separation.
3. NVC practice: observation / feeling / need / request separation.

# Open Gaps — Кантовская основа ННО / гигиена предикации

Status: active gaps / non-canon
Opened: 2026-05-28

## Core Gaps

1. **Kant textual grounding**
   - Verify the exact passages and conceptual boundaries in Kant.
   - Separate the user's interpretation from Kant's own claims.
   - Clarify whether the strongest anchor is `Critique of Judgment`, `Critique of Pure Reason`, or the broader critical method.

2. **NVC textual grounding**
   - Verify the observation / feeling / need / request structure from primary NVC sources.
   - Avoid turning NVC into a purely Kantian derivative.

3. **Predicate type taxonomy**
   - Distinguish object predicates, perception predicates, relational predicates, affective predicates, moral predicates, motive attributions, and requests.
   - Determine which predicate shifts actually produce conflict escalation.

4. **Subjective-universal tension**
   - In Kantian aesthetics, the judgment is subjective in source but carries a claim to communicability or universal assent.
   - Clarify whether interpersonal accusations have an analogous structure or only superficially similar grammar.

5. **Relation to SynTax and R Theory**
   - Map FACT / INTERPRETATION / HYPOTHESIS discipline to NVC categories without flattening them.
   - Determine whether predication hygiene is an R3 operator, an R2 communication procedure, or a mixed R3->R2 practice.

6. **Application boundary**
   - Some statements about people can be justified stable behavioral claims.
   - The module must not force every criticism into purely subjective feeling language.

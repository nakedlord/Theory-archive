# Lomekwi proto-operation source card

## Source

Primary source:

- Sonia Harmand et al. "3.3-million-year-old stone tools from Lomekwi 3, West Turkana, Kenya." Nature 521, 310-315, published 2015-05-20.
- DOI: `10.1038/nature14464`
- URL: https://www.nature.com/articles/nature14464

## Source-level claims used

The source reports Lomekwi 3 as a 3.3-million-year-old archaeological site in West Turkana, Kenya, with in situ stone artefacts associated with Pliocene hominin fossils in a wooded palaeoenvironment.

The source states that Lomekwi 3 knappers showed a developing understanding of stone fracture properties and combined core reduction with battering activities.

The source proposes the term `Lomekwian` for the assemblage and states that it predates the Oldowan by about 700,000 years.

## R Theory use

This source supports the `proto-operation` layer of the action-symbol archaeology check.

R Theory reading:

```text
percussive action / battering + core reduction -> early stabilized operation, but not yet clear action-symbol grammar
```

## Interpretation

Lomekwi is stronger than accidental stone breakage and weaker than an Oldowan-like action-symbol threshold.

It supports:

```text
R1+ / R2- proto-operation
```

It does not yet support:

- stable portable action category;
- recombinable action grammar;
- symbolic carrier;
- R3.

## Use in current hypothesis

Lomekwi should be used as the lower bound of the action-first abstraction chain:

```text
operation begins before action-symbol threshold
```

## Non-claims

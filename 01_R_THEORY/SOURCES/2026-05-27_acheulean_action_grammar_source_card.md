# Acheulean action grammar source card

## Source

Primary source:

- Christopher J. Lepre et al. "An earlier origin for the Acheulian." Nature 477, 82-85, published 2011-08-31.
- DOI: `10.1038/nature10372`
- URL: https://www.nature.com/articles/nature10372

## Source-level claims used

The source reports Acheulian stone tools at Kokiselei 4 in West Turkana, Kenya, dated to 1.76 Ma.

The source characterizes Acheulian technology by shaped bifacial stone tools and notes that Oldowan and Acheulian artefacts co-occur in the West Turkana record, implying multiple stone-tool-making behaviours rather than a simple replacement sequence.

## R Theory use

This source supports Acheulean as the first strong candidate for an `action grammar` layer.

Working operational sequence:

```text
select core -> establish edge -> bifacial thinning -> shaping -> correction
```

The important theoretical shift is from producing a useful sharp edge to organizing multiple operations around a stable form target.

## Use in current hypothesis

It does not by itself prove R3. R3 would require clearer evidence for explicit comparison, causal rule correction, cross-domain transfer, or reflective error handling.

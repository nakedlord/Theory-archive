# PMv action symbols as evidence for action-first abstraction

## Status

Status: `non_canon_evidence`.

This note records an evidence candidate for R Theory. It does not promote a canon change.

## Evidence anchor

A 2026 Nature paper reports that macaques can learn and recombine drawing-like action primitives, and that ventral premotor cortex (PMv) contains neural population activity encoding planned actions with invariance, categorical structure, and recombination.

Source record: `01_R_THEORY/SOURCES/2026-05-25_pmv_action_symbols_nature_source.md`.

## R Theory interpretation

The paper is useful because it supports an action-first route into abstraction:

```text
action -> stable motor primitive -> invariant action category -> recombinable action-symbol -> external trace / artifact / sign
```

This is compatible with an R Theory reading in which early abstraction need not begin as language, explicit proposition, or conscious symbol manipulation. A symbolic-like unit can first appear as a stabilized action operator.

## Fit with R-levels

### R1 / repeated action

At the lower bound, repeated movement may solve a task without detachable structure.

### R2 / compositional operation

The PMv result is closer to an R2-compatible layer: learned actions become reusable parts that can be composed into new sequences.

This does not require full reflective explanation of the action, only stabilization and recombination of action primitives.

### R3 / reflective abstraction

The evidence does not establish R3. R3 would require a further level: explicit comparison, error analysis, causal interpretation, or transfer of the operation as a rule across domains.

## Archaeology implication

For artifact interpretation, this strengthens a methodological rule:

Do not read early artifacts only as objects or shapes. Read them also as external traces of stabilized operations.

An artifact becomes theoretically important when the underlying operation shows signs of:

1. invariance across material, size, or context;
2. categorical stability as a type of action;
3. recombination into new sequences or tool-making grammars.

This may help distinguish:

- repeated use;
- operational tradition;
- compositional technique;
- later symbolic residue.

## Relation to existing R Theory formula

Existing R Theory status distinguishes:

```text
function -> significance -> symbol -> transferable distinction -> concept
```

The PMv evidence suggests a complementary pre-symbolic path:

```text
operation -> action-symbol -> external trace -> symbolic carrier
```

Open question: whether these two paths converge through the same transition from functional operation to retained significance after original function weakens or disappears.

## Limits / non-claims

This note does not claim:

- PMv is the general brain region of abstract thought;
- macaques have human symbolic recursion;
- this directly explains language;
- this proves any specific archaeological stage;
- this is already R Theory canon.

The safer claim is narrower: primate action planning can contain symbol-like motor units before language-level symbolic systems.

## Next test

Compare the action-symbol criteria against archaeological sequences:

- Lomekwi / Oldowan: repeated operation vs reusable action category;
- Acheulean: whether handaxe production implies stronger operational invariance and recombinable action grammar;
- ochre / burial: whether retained significance after loss or weakening of function creates the symbolic carrier layer.

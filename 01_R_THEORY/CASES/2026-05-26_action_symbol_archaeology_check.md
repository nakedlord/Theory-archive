# Action-symbol archaeology check

## Status

Status: `non_canon_case_check`.

This case check extends the PMv action-symbol evidence note into an archaeology-facing test. It does not promote a canon change.

Source anchor: `01_R_THEORY/SOURCES/2026-05-25_pmv_action_symbols_nature_source.md`.

Evidence note: `01_R_THEORY/EVIDENCE/2026-05-25_pmv_action_symbols_evidence_note.md`.

## Tested criteria

The PMv/action-symbol frame gives three criteria for recognizing an operation that has become symbol-like at the action level:

1. invariance across low-level implementation details;
2. categorical stability as a type of action;
3. recombination into new sequences or action grammars.

R Theory archaeology translation:

```text
artifact is not only an object; it is an external trace of a stabilized operation
```

## Interpretation

### Lomekwi

Lomekwi supports intentional operation and possibly an early operational schema, but it does not yet show enough evidence for a stable, recombinable action grammar.

Working status:

```text
R1+ / R2- proto-operation
```

### Oldowan

Oldowan is a stronger candidate because the operation can be read as a stable category:

```text
strike / detach -> sharp edge
```

This begins to satisfy the action-symbol threshold: a repeated action becomes a portable technological category rather than a one-off movement.

Working status:

```text
action-symbol threshold candidate
```

### Acheulean

Acheulean bifacial shaping strengthens all three criteria. It implies a sequence in which selected operations are ordered toward a form target:

```text
select core -> establish edge -> bifacial thinning -> shaping -> correction
```

This looks less like a single operation and more like an action grammar.

Working status:

```text
R2+ / action grammar candidate
```

### Ochre / burial / symbolic residue

Ochre and burial-like material are not the beginning of the action-symbol layer. They belong to a later layer where operations, materials, traces, and significance can stabilize into symbolic carriers.

Working status:

```text
symbolic carrier layer
```

## Hypothesis status after check

The check supports the architecture of the hypothesis in a limited but meaningful sense:

```text
operation -> stabilized operation -> portable action category -> recombinable action grammar -> external trace -> symbolic carrier
```

Supported:

- abstraction can begin as stabilized operation, not language;
- early artifacts should be read as traces of operations, not only objects;
- Oldowan is a plausible lower bound for an action-symbol threshold;
- Acheulean is a stronger candidate for action grammar.

Not established:

- full R3;
- language origin;
- developed symbolic meaning;
- direct proof of any specific mental state from a single artifact class.

## Canon boundary

This case check strengthens the non-canon evidence layer. It does not update `CANON.md`.

Canon promotion would require:

1. source cards for the archaeology layers;
2. a more exact comparison across Lomekwi, Oldowan, Acheulean, and ochre/burial evidence;
3. separation of artifact morphology, operational sequence, teaching/transmission, and symbolic significance;
4. review against the R Theory canon checklist.

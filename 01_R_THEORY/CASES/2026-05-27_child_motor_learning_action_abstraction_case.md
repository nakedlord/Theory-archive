# Child motor-learning action abstraction case

## Status

Status: `candidate_support`.

This is a non-archaeological case supporting the candidate patch:

- `01_R_THEORY/PATCHES/2026-05-27_action_first_abstraction_candidate_patch.md`

It does not update canon.

## Purpose

The archaeology sequence can overfit the model to stone tools. This case checks whether action-first abstraction also makes sense in child action learning.

The target question:

```text
Can an action become a portable goal-directed unit before it becomes explicit language, symbol, or concept?
```

## Source anchors

Primary source candidates for later full source cards:

- Andrew N. Meltzoff. "Understanding the intentions of others: Re-enactment of intended acts by 18-month-old children." Developmental Psychology, 1995.
- György Gergely, Harold Bekkering, Ildikó Király. "Rational imitation in preverbal infants." Nature, 2002.

These sources are used here at case-anchor level, not as a full literature review.

## Case logic

Child imitation studies are relevant because they separate copying surface movement from extracting an action goal or action unit.

If a child reproduces the intended act rather than merely the observed bodily trajectory, then the child is not only copying motion. The child is extracting a portable action structure:

```text
observed movement -> inferred action goal -> reproduced action pattern
```

This supports the action-first abstraction layer at a non-archaeological scale.

## Mapping to action-first abstraction

| Layer | Child motor-learning analogue | R Theory reading |
|---|---|---|
| Repeated action | child copies visible movement | R1-like imitation |
| Stabilized operation | child repeats action as a goal-directed routine | proto-operation |
| Portable action category | child recognizes same action across different bodily details or contexts | action-symbol-like layer |
| Recombination / grammar | child combines learned routines into longer scripts | R2-compatible action composition |
| External trace | action produces a visible result in object/world | trace of operation |
| Symbolic carrier | later, action/result can be named, narrated, normed, or ritualized | symbolic / instruction layer |

## What this supports

This case supports the candidate claim that action abstraction can precede language-level symbolic abstraction.

It supports:

```text
action can be abstracted from concrete movement
```

and therefore strengthens:

```text
operation -> portable action category
```

## What this does not support

This case does not prove:

- archaeology thresholds;
- R3;
- language origin;
- symbolic culture;
- that infant imitation and stone-tool production are the same mechanism;
- that all goal imitation is action-symbol in the strict PMv sense.

## Why it matters for the candidate patch

The candidate patch should not depend only on artifacts. Child motor-learning gives a living developmental analogue of the same structural distinction:

```text
motion/form != operation/goal
```

This mirrors the archaeology distinction:

```text
artifact form != trace of operation
```

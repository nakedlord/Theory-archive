## Purpose

The table separates layers that are easy to collapse when interpreting artifacts:

- morphology;
- operational sequence;
- portability / category;
- recombination / grammar;
- transmission;
- significance / symbolic residue.

The goal is to prevent direct jumps from artifact form to R-level or symbolic meaning.

## Minimal interpretation

The sequence is not a single universal ladder. It is a layered diagnostic frame:

```text
proto-operation -> portable action category -> recombinable action grammar -> symbolic carrier
```

## Strongest current support

1. Lomekwi supports the existence of pre-Oldowan proto-operation.
2. Oldowan is the first strong candidate for portable action category.
3. Acheulean is the strongest current candidate for recombinable action grammar.
4. Ochre/graphic marks and burial-like contexts belong to symbolic carrier formation, not the origin of action-symbols.

## Key safeguards

### Artifact morphology is not cognition

Form alone does not prove an R-level.

### Tool tradition is not automatically symbol

A stable operation may be a portable action category without symbolic meaning.

### Action grammar is not automatically R3

Acheulean-like grammar can be R2/R2+ unless reflective comparison, causal rule correction, cross-domain transfer, or explicit error analysis is independently supported.

### Symbolic carrier is not automatically concept

A trace can carry significance before it becomes a transferable conceptual distinction.

## Use in canon review

If the candidate patch reaches canon, this table should remain outside canon as support material.

The canon should only preserve the compact distinction:

```text
artifact as object/form != artifact as external trace of stabilized operation
```

and the limited formula:

```text
operation -> stabilized operation -> portable action category -> recombinable action grammar -> external trace -> symbolic carrier
```

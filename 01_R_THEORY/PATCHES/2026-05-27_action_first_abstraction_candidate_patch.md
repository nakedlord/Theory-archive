# Action-first abstraction candidate patch

## Status

Status: `candidate`.

This is a candidate patch for R Theory. It does not update `CANON.md`.

## Candidate distinction

Early abstraction can begin as a stabilized action layer before it becomes explicit symbol, language, or concept.

Core distinction:

```text
artifact as object/form != artifact as external trace of stabilized operation
```

Candidate formula:

```text
operation -> stabilized operation -> portable action category -> recombinable action grammar -> external trace -> symbolic carrier
```

## Before patch

R Theory v3.2 currently emphasizes the symbolic/conceptual path:

```text
function -> significance -> symbol -> transferable distinction -> concept
```

This is useful for symbol-to-concept development, but it leaves the lower action layer under-specified.

Problem:

Without an action-first layer, early artifacts are too easily read as either:

1. objects/forms;
2. tools/functions;
3. already-symbolic traces.

The missing intermediate is the operation becoming portable before it becomes symbolically meaningful.

## After patch

Add a lower operational-abstraction layer beneath the existing symbolic/conceptual chain.

Combined architecture:

```text
operation
-> stabilized operation
-> portable action category
-> recombinable action grammar
-> external trace
-> functional/significance retention
-> symbol
-> transferable distinction
-> concept
```

This does not replace the existing v3.2 formula. It adds a pre-symbolic action abstraction layer that can feed into it.

## Why this is needed

The PMv/action-symbol evidence suggests that primate action planning can encode action units with:

1. invariance;
2. categorical structure;
3. recombination.

The archaeology-facing case check then maps those criteria onto a plausible sequence:

```text
Lomekwi -> Oldowan -> Acheulean -> ochre/burial
```

Working interpretation:

```text
proto-operation -> action-symbol threshold -> action grammar / R2+ -> symbolic carrier layer
```

This gives R Theory a clearer bridge between:

- neural/action abstraction;
- archaeological artifact production;
- later symbolic carrier formation.

## R-level interpretation

### R1 / repeated action

Repeated action may solve a task without becoming a portable action category.

### R1+ / R2- proto-operation

Lomekwi-like behavior can be read as proto-operation: intentional percussive activity and early fracture-property handling without established action grammar.

### R2 / portable action category

Oldowan-like sharp-edge production is a candidate for the action-symbol threshold:

```text
strike / detach -> sharp edge
```

This is not symbolic meaning. It is a portable action category.

### R2+ / action grammar

Acheulean-like bifacial shaping is a stronger candidate for recombinable action grammar:

```text
select core -> establish edge -> bifacial thinning -> shaping -> correction
```

### R3 boundary

This patch does not establish R3.

R3 would require stronger evidence for explicit comparison, causal rule correction, cross-domain transfer, reflective error analysis, or self-positioning relative to the operation.

## Relation to symbolic carrier formation

Ochre/graphic marks and burial-like contexts should not be treated as the beginning of action-symbol formation.

They are better read as a later layer where:

```text
operation / material practice / bodily or funerary handling -> retained significance -> symbolic carrier
```

This protects the model from collapsing tool operation and symbolic meaning into one step.

## Limits

This patch does not claim:

- PMv is a general center of abstract thought;
- macaques have human symbolic recursion;
- Oldowan definitively equals action-symbol layer;
- Acheulean definitively equals R2+;
- ochre/burial proves full symbolic meaning;
- any of this proves R3, language, religion, or concept formation;
- artifact morphology alone gives direct access to mental state.

## Risks

### 1. Archaeology overreach

Risk: reading cognitive level directly from artifact form.

Mitigation: require separation between morphology, operational sequence, transmission, and significance.

### 2. Symbol inflation

Risk: calling every stable operation a symbol.

Mitigation: use `action-symbol` only for portable action categories with invariance, category, and recombination evidence; reserve `symbolic carrier` for retained significance.

### 3. R2/R3 confusion

Risk: treating action grammar as reflective abstraction.

Mitigation: mark action grammar as R2/R2+ unless reflective comparison/correction evidence appears.

### 4. One-line evolutionary ladder

Risk: treating Lomekwi -> Oldowan -> Acheulean -> ochre/burial as a single inevitable ladder.

Mitigation: read it as a test sequence of layers, not a strict universal lineage.

# M5 / Z_e Entanglement Test

## Current Formulation

The current working test reformulates the M5 idea as a hidden relational dimension / connectivity layer for entanglement:

```text
M5 = ordinary spacetime + hidden entanglement-connectivity axis Z_e
```

`Z_e` is not an ordinary spatial road and not a traversable signal channel. It is a candidate physical layer in which entangled systems can remain relationally close even when they are far apart in visible 3D space.

The hypothesis is not yet canon. It is a test frame for whether entanglement can be understood as physical relation without local readable marks and without controllable superluminal signaling.

## Why It Appeared

This idea emerged from the following chain:

1. Bell/CHSH excludes simple local hidden-value tables.
2. Entangled and unentangled local subsystems can look identical under local measurement.
3. Therefore the physical difference must live in the joint relation, not in either endpoint alone.
4. If the relation is physical but not a 3D signal channel, it suggests hidden topology / additional connectivity.
5. This returns to the M5/Z_e intuition: distance in visible 3D may not equal distance in relational connectivity.

## Core Requirements for Z_e

`Z_e` must satisfy:

```text
1. Local invisibility:
   one particle alone must not reveal whether it is entangled.

2. Joint constraint:
   A-B measurements must show correlation structure stronger than local hidden tables.

3. No signaling:
   Z_e must not allow controllable bit transmission.

4. One-shot / decoherence behavior:
   strong measurement or environmental interaction must break or redistribute A-B relational state.

5. Monogamy:
   if A is maximally entangled with B, A cannot be equally maximally entangled with C.

6. Rewiring:
   entanglement swapping must be explainable as a redistribution or re-routing of relational state.

7. Tsirelson bound:
   M5/Z_e must explain why CHSH can exceed 2 but not exceed 2√2.
```

## What This Solves

The test tries to avoid two weak positions:

1. `Entanglement is only knowledge` — too weak if Bell correlations are physically real.
2. `A sends a signal to B` — too strong and conflicts with no-signaling.

M5/Z_e offers a third option:

```text
A and B are not independently local in the hidden relation-space, but the relation is not a message channel.
```

## Failure Modes

1. Beautiful-metaphor failure

If Z_e does not produce calculations or testable distinctions, it is only a metaphor.

2. Naive wormhole failure

If the hidden connection is traversable, it predicts faster-than-light signaling.

3. Local-hidden-variable collapse

If Z_e only stores local pre-written answers, Bell/CHSH still blocks it.

4. Unlimited-link failure

If Z_e allows unlimited maximal links, it violates monogamy.

5. PR-box failure

If Z_e is unconstrained, it may predict correlations stronger than quantum mechanics.

6. Ether failure

If Z_e behaves like ordinary hidden matter/medium, it should create detectable drag, preferred frame, or extra scattering.

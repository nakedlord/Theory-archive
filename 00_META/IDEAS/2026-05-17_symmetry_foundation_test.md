## Test Target

Candidate mathematical apparatus:

```text
Symmetry / group theory / transformations and invariants
```

Core ontological inversion:

```text
What if physics is not fundamentally about objects, but about admissible transformations and what remains invariant under them?
```

## Primitive Bet

The primitive is not a thing, but a transformation field:

```text
Reality = a structured set of possible transformations.
Object = stable invariant pattern across transformations.
Law = constraint on admissible transformations.
Conserved quantity = invariant produced by a transformation symmetry.
```

## Immediate Strengths

Symmetry is an unusually strong candidate because it already touches several deep physical layers:

1. Laws are often expressed as invariance under transformations.
2. Conserved quantities follow from continuous symmetries in modern physics.
3. Particle types can be classified by symmetry representations.
4. Gauge symmetries structure fundamental interactions.
5. Spacetime structure is tightly tied to transformation groups.
6. Broken symmetries explain phase transitions and mass-generation-like mechanisms in field theory.

This makes symmetry more than a calculational convenience. It behaves like a generator of physical grammar.

## Ontological Reading

If symmetry is fundamental, then physical identity is secondary.

A physical object is not first a substance with properties. It is a stable answer to the question:

```text
What remains the same across allowed transformations?
```

In this frame:

- a particle is a stable representation of transformation structure;
- a field is a structured assignment over a transformation-sensitive domain;
- a conservation law is an invariant constraint;
- interaction is a constrained transformation between states;
- measurement is a symmetry-breaking or distinction-stabilizing operation;
- classical objects are high-stability invariant bundles.

## Main Provisional Result

Symmetry is not yet shown to be a complete foundation by itself.

It is a high-rank candidate layer because it helps explain why laws, conservation, classification, interactions, and spacetime transformations have the form they do.

But this remains a conceptual inference, not a proof.

The provisional gap is:

```text
Symmetry appears to constrain possible physics better than it generates actuality.
```

It tells us what must remain invariant under allowed transformations, but it has not yet been shown to generate:

- why there is a state space at all;
- why a particular state is actual;
- why probability appears;
- why measurement stabilizes one outcome;
- why time has an experienced/thermodynamic direction;
- why a specific action/Hamiltonian is selected.

## Failure Mode Candidate

### Constraint Without Actualization

Symmetry is excellent at defining equivalence, invariance, and conservation. But a world cannot be only equivalence classes. It must also include actual transitions, boundary conditions, state selection, and broken symmetry.

This is a candidate failure mode, not a proved impossibility.

## Stronger Reframing

Instead of:

```text
Reality is symmetry.
```

Use:

```text
Reality may require a transformation/invariant layer, but this layer may need coupling to a state-selection and distinction-stabilization layer.
```

This suggests a possible composite foundation:

```text
Transformations + invariants + state space + distinction/measurement + information loss/coarse-graining
```

A real version of this test should begin with a minimal formal candidate, for example:

```text

Mark every imported structure as non-derived.
```

Only after that can the candidate be ranked mathematically.
